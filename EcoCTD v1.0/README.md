# ecoCTD-open-project

## EcoCTD v1.0

This repository includes all the files necessary to make the EcoCTD v1.0.

### EcoCTD v1.0 unique characteristics
* RINKO ODO sensor
* Requires both lower and upper tube
* ODO and chl/bbp sensor in the same section tube
